# {{project_name}}

This project was scaffolded using [Nuda](https://github.com/azatdavliatshin/nuda) — a minimalist, self-hosted framework for full-stack web apps using native HTML, JS, and CSS.

## 📁 Project Structure

- `pages/` – static HTML pages (SSG)
- `routes/` – server-side handlers (SSR / API)
- `public/` – static assets (served as-is)
- `nuda.config.json` – basic configuration
- `package.json` – CLI commands
- `README.md` – you’re reading it

## 🚀 Available Commands

- `nuda dev` – start development server with live reload
- `nuda build` – generate static output (SSG)
- `nuda deploy` – deploy via configured provider

## 🧭 Philosophy

Nuda encourages:

- Zero-runtime, file-based architecture
- Full control over rendering

> Less magic — more control.
